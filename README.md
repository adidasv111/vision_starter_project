# Starter Project
This is a starter project for:
    - PCL + OpenCV project using CMake
    - Python app with requirements.txt

## PCL build
```bash
mkdir build
cd build
cmake ..
make
```

## Python build
```bash
pip -r requirements.txt
./main.py
```

## PCD Visualizer
Load all pcd files from given folder and visualize them one by one.
Path should end with `/`
